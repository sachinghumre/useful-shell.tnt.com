# git

## config
```
# 系统:/etc/gitconfig
# 用户:~/.gitconfig 或 ~/.config/git/config

# 每一个git的提交都需要记录username和email，需要设置
git config --global user.name "xxx"
git config --global user.email "xxx@example.com"

# 检查配置信息
git config --list
```

```
# 查看命令使用
```
